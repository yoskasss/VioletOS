VioletOS
Eğitim amaçlı x86 32-bit işletim sistemi — C ve Assembly
v1.0.0 x86 32-bit MIT lisansı Türkçe Q klavye Violet IDE Violet Dili
GitHub Ekran Görüntüleri Kurulum Violet IDE Violet Dili Döküman

Nedir?

VioletOS; GRUB multiboot kernel, kalıcı dosya sistemi, Violet IDE ve kendi programlama dili içeren tam kapsamlı bir eğitim amaçlı işletim sistemidir. Her şey C ve Assembly ile sıfırdan yazılmıştır.

İşletim sistemi geliştirmeyi öğrenmek isteyenler için açık kaynaklı bir proje.

Özellikler

  • GRUB multiboot kernel (x86 32-bit protected mode)
  • GDT, IDT, PIC IRQ yönetimi
  • VGA text modu (mor tema, 80×25) — donanım ve yazılım imleci
  • PS/2 klavye sürücüsü — Türkçe Q düzeni (ğ, ü, ş, ı, ö, ç…)
  • PIT timer (100 Hz) — uptime, gecikmeler
  • ATA PIO disk sürücüsü — kalıcı dosya depolama
  • VioletFS — basit in-kernel dosya sistemi (64 giriş, 4 KB/dosya)
  • İnteraktif shell — 15+ komut, yanıp sönen blok imleci
  • Violet IDE — satır numaraları, aktif satır vurgusu, yazılım imleci
  • Violet dili — .v uzantılı betik dosyaları için yorumlayıcı

Kurulum & Derleme

$ sudo dnf install gcc nasm make grub2-tools xorriso \
                  qemu-system-x86 glibc-devel.i686 libgcc.i686
$ sudo apt-get install gcc-multilib nasm make \
                    grub-pc-bin xorriso qemu-system-x86

Derleme

$ chmod +x build.sh && ./build.sh
# veya
$ make iso
→ build/violetos.iso oluşturuldu

QEMU ile çalıştırma

$ make run
# kalıcı disk ile
$ make disk-image
$ make run-disk █

USB'ye yazma (gerçek donanım)

$ sudo dd if=build/violetos.iso of=/dev/sdX bs=4M status=progress conv=fsync

⚠ /dev/sdX yerine kendi USB aygıtınızı yazın. Sürücü silinir. UEFI makinelerde Legacy/CSM modunu etkinleştir.

Violet IDE

Ana menüden veya shell'de ide komutuyla açılır. Tam ekran, .v dosyaları için satır numaralı editör.

Düzen (80×25 metin)

Satırİçerik
0Başlık çubuğu (magenta) — dosya adı, kısayollar
1Yatay çizgi
2–19Editör — satır cetveli + kod alanı
20Yatay çizgi
21Komut çubuğu — : open | save | new | run
22Durum çubuğu — mesajlar, Ln X/Y

Kısayollar

Kısayolİşlem
Ctrl+SVioletFS'ye kaydet
Ctrl+RKodu çalıştır
Ctrl+QÇık (değişiklik varsa otomatik kaydeder)
:Komut modu (open dosya.v, save, new, run)
Ok tuşlarıİmleç hareketi

Aktif satır; mor cetvel + mavi arka plan ile vurgulanır. İmleç yazılım tabanlı beyaz bloktur.

Violet Dili

Dosyalar .v uzantısı kullanır. IDE'den (Ctrl+R) veya shell'den (run program.v) çalıştırılır.

Örnek

// Violet ornek print "Merhaba VioletOS!" set ad "Violet" set sayi 10 print ad + " dili" if sayi > 5   print "Buyuk" else   print "Kucuk" end set i 0 while i < 3   print i   set i i + 1 end
